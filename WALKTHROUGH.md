# 漢字書き取りアプリ 実装完了レポート

## 実装された機能
- **タイトル画面**: 「1ねんせい」などのボタンで学年を選択してスタートします。
- **バトル（クイズ）画面**: 
  - `public/data/grade1.csv` から問題を読み込みます。
  - 問題文を表示し、対象の単語（読み）を強調表示します。
  - **書き取りエリア**: `hanzi-writer` を使用したインタラクティブな漢字書き取りエリアです。
    - 正解の画: 自動で進行します。
    - 不正解/書き順ミス: 振動のような視覚フィードバック（またはログ）を行い、ヒントを表示します。
  - **進行**: 正解すると自動的に次の問題へ進みます。
- **結果発表画面**: スコアを表示し、全問正解などの場合に紙吹雪でお祝いします。
- **技術スタック**:
  - React + TypeScript (Vite)
  - Tailwind CSS v4
  - Hanzi Writer (漢字書き取りライブラリ)
  - PapaParse (CSV解析)

## 動作確認・検証方法

1. **開発サーバーの起動**:
   ```bash
   npm run dev
   ```
2. **アプリを開く**:
   - ブラウザで `http://localhost:5173` にアクセスします。
3. **学年選択**:
   - **1ねんせい** をクリックします。
4. **クイズに挑戦**:
   - 問題文を読みます（例：`[にち]ようび`）。
   - 下のボックスに漢字（例：`日`）を書きます。
   - 正しく書けると次の問題に進みます。
   - もし間違えると、コンソールにミスが表示されたり、再度入力を求められます。
5. **終了**:
   - 全5問（デフォルト設定）を解き終わると、結果画面が表示されます。
   - 「もういっかい」でリトライ、「もどる」でタイトルに戻れます。

## 備考
- **問題データ**: `public/data/grade1.csv` にCSV形式で保存されています。このファイルを編集・追加することで問題を増やせます。
- **漢字データ**: CDNから動的に取得しています。新しい漢字を表示するにはインターネット接続が必要です。
